<h1 align="center">CanvasPads</h1>

<p align="center">
An open source drawing and painting app for making arts.
</p>

## Building

### Install wasm-bindgen-cli

```sh
$ cargo install -f wasm-bindgen-cli
```

### Build `canvaspads-web`

```sh
$ cd ./crates/canvaspads-web
$ ./build.sh
```
